Powerful Features

Git-Driven Deployments

Define your entire infrastructure in Git. Konta watches for changes and deploys automatically.

git push ‚Üí konta deploys ‚Üí done

Atomic Deployments

Zero-downtime updates using symlink switching. If something goes wrong, automatic rollback.

ln -s releases/v2 current

Smart Lock Management

Prevents concurrent deployments. Only one reconciliation at a time, every second is safe.

flock /var/run/konta.lock

Pre/Post Hooks

Run custom scripts before deploy, after success, or on failure. Integrate with your workflow.

hooks/pre.sh hooks/success.sh

Multi-Server Ready

Manage multiple servers from one Git repository. Each server points to its own folder.

path: spb/apps OR path: ger/apps

Label-Based Management

Only manages containers labeled with konta=true. Your existing containers stay safe.

label: konta.managed=true
